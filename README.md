HealthGuard is a web-based Disease Prevention & Monitoring platform that enables users to track vital signs, assess health risks, and receive preventive alerts for early detection.

## Features
- User & Admin authentication
- Vitals tracking (BP, heart rate, sugar level, temperature)
- Disease risk assessment (Low / Medium / High)
- Health alerts & notifications
- Admin dashboard & analytics
- Secure PHP PDO backend

## Tech Stack
- HTML, CSS, JavaScript
- PHP (PDO)
- MySQL

## Setup Instructions
1. Clone the repository
2. Import `database/healthguard.sql` into MySQL
3. Configure database credentials in `config/db.php`
4. Run on a local server (XAMPP, WAMP, or Laragon)

> All health data used is mock data for demonstration purposes.
